# LeStar-Zs-Speak-the-Beat
This is a metronome that speaks the count in lieu of just clicks. This allows for practice sessions to feel a little friendlier and less of a chore. It is modifiable to many beats and rhythms, and is useable in 32 langages. It is very simple to set up, modify to particular specifications, and use in private practice sessions and public performances. It is designed for use on a mobile phone.
